DEFINITION

DOM : Document Object Model

Lorsqu'une page web est chargee, le navigateur cree un Model d'Objet du Document( Document Object Mode / DOM en anglais)

Le DOM HTML est contruit comme un arbre d'objets

Avec le DOM HTML on peut alors utiliser Javascript pour creer du HTML dynamique.

Javascript est capable de :

  • Creer, modifier, supprimer les elements et attributs HTML
  • Creer, modifier, supprimer les styles CSS
  • Reagir a des evenement HTML existant
  • Creer de nouveaux evenement HTML

Ce qu'il faut retenir c'est que le DOM HTML est aujourd'hui le standard pour manipuler les element HTML (recuperation, ajout, suppression, modification)
